\section{Literature Review}

\subsection{Recommendation and Matching Systems}
Recommendation systems are used to suggest suitable people, items, or services based on user preferences and behavior. In social applications, they help users find people with similar interests and lifestyles. TRIBAL uses this concept to recommend compatible roommates and activity partners. Since a large dataset is not available for this minor project, a rule-based weighted matching system is used instead of machine learning.

\subsection{Roommate Matching Systems}
Existing roommate matching systems help users find roommates based on preferences such as budget, food habits, cleanliness, sleep schedule, and lifestyle. These systems show that roommate compatibility depends on more than just room availability. However, most roommate matching systems only focus on finding rooms or roommates and do not include activity discovery, events, or community-building features.

\subsection{ Event and Activity Recommendation}
Event recommendation systems help users discover activities based on interests, location, or user behavior. These systems are useful for connecting people with similar hobbies in real life. TRIBAL follows this idea by allowing users to create, browse, and join local events. However, instead of using complex location tracking, TRIBAL uses interest-based event discovery, which is simpler and more suitable for this project.

\subsection{Review of Existing Platforms}
Platforms like Roomster, Roomi, and SpareRoom focus mainly on roommate searching, while Meetup and Bumble BFF focus on social connection and activity discovery. These platforms solve only one part of the problem. In Nepal, many users still depend on Facebook groups or personal contacts, which lack proper compatibility checking, verification, and structured matching.

\subsection{Research Gap}
From the reviewed systems, it can be seen that existing platforms usually focus separately on roommate matching, friend discovery, or event participation. There is a lack of a single mobile application that combines compatible roommate matching, activity partner discovery, local events, and secure chat. TRIBAL addresses this gap by using a weighted compatibility scoring algorithm based on lifestyle and interest-related onboarding answers.
